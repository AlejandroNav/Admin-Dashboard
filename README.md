# Admin Dashboard

A clean and modern admin dashboard interface built with **HTML** and **CSS Grid** as part of **The Odin Project**.

## Overview

This project is a static admin dashboard layout focused on practicing:

- CSS Grid
- Nested grids
- Semantic HTML
- Dashboard-style UI structure
- Reusable layout patterns

The goal was to recreate a dashboard-style interface with a sidebar, header, project cards, announcements, and trending section.

## Features

- Sidebar navigation with branding and external GitHub repository link
- Header with:
  - Search bar
  - Notifications area
  - User profile section
  - Action buttons
- Project cards section
- Announcements panel
- Trending panel
- Responsive adjustments for smaller screens
- Custom typography and styling
- Hover effects and accessible interactive elements

## Built With

- HTML5
- CSS3
- CSS Grid
- Google Fonts

## What I Practiced

During this project, I worked on:

- Building a full dashboard layout with **CSS Grid**
- Using **nested grids** for smaller layout sections
- Structuring content with **semantic HTML**
- Improving visual hierarchy with spacing, typography, and cards
- Creating reusable UI sections like navigation, panels, and action buttons
- Making a static project ready for **GitHub Pages**
